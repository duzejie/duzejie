## redis

Redis（全称：Remote Dictionary Server 远程字典服务）是一个开源的使用 ANSI C 语言编写、支持网络、可基于内存亦可持久化的日志型、Key-Value 数据库，并提供多种语言的 API

#### 数据结构

String: 字符串
Hash: 散列
List: 列表
Set: 集合
Sorted Set: 有序集合
